# ECMS
## Electro Corparation Music Software
### 
ECMS is a lightweight music creation tool developed in Java, for portability. <br>
It features a wide range of instrument options, and an easy to use interface. 
### File format
#### .sps
The sps file format is an abstraction built ontop of the Java Midi player, that allows for easy saving/loading <br>
of music.
